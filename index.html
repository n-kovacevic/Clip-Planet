Security issues

Most of the scripts don't check if any users is logged in. Therefore if anybody access upload script while logged out script will upload video and user uploading video will be set to "admin". It is possible to upload only .png files as video thumbnail. Uploading any other file will result in video not having thumbnail. It might be possible to register user without a password. And many other issues I do not plan on fixing in the future.
